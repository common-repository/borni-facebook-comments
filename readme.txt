This plugin implements facebook comments widgeton your wordpress blog.
It appends the facebook comments widget at the bottom of each post.

== Description ==
 
To install just download the zip locally, and click
Plugins > New > Upload and upload the zip.
Once Uploaded click activate.

Upon activated a new menu 'FB Comments' will appear on your
admin dashboard. From the admin panel you can customize the following
settings:

For the widget to work the plugin imports the OFFICIAL javascript code from:
https://developers.facebook.com/docs/plugins/comments/

To get an app id, first you must create an APP at:
https://developers.facebook.com/apps/

You can add an facebook app id to moderate comments.
You can change the widget language so it can be implemented in a different language other than english.
You can change the number of displayed comments. Default is 10.
